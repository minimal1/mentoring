## 학습 진도

- [ ] 1단계: JavaScript 깊이 파고들기
- [ ] 2단계: Node.js 기본 다지기
- [ ] 3단계: Express.js로 API 서버 구축
- [ ] 4단계: 데이터베이스 연동
- [ ] 5단계: 보안 강화
- [ ] 6단계: 배포 및 운영
- [ ] 7단계: 실전 프로젝트 참여 및 심화 학습
